# Postcard Landing Page Redesign

## Goal
Replace the generic SaaS landing page with an animated, show-don't-tell experience that demonstrates the AI building an email in real time.

## Files to modify
- `app/pages/index.vue` — full rewrite (~400 lines)
- `app/assets/css/main.css` — append ~25 lines of new animation classes

## New section structure

| # | Section | What it does |
|---|---------|-------------|
| 1 | **Animated Hero** | 3-pane studio mockup where typing a prompt builds an email block-by-block in real time |
| 2 | **Trust Bar** | Placeholder logo row + one testimonial quote |
| 3 | **Feature Showcase** | 3 alternating large visual sections (AI chat, cross-client rendering, API) + 3 compact cards |
| 4 | **Template Gallery** | Horizontal scroll of 6 email template preview cards with accent colors |
| 5 | **Final CTA** | Aurora-backed window with stronger copy |

## 1. Animated Hero — the "wow" moment

Same `pc-window` macOS mockup, but center pane shows real email blocks appearing one by one, right pane shows chat messages being typed.

### Animation sequence (Product Launch email, loops every ~26s)

| Time | Chat | Email preview |
|------|------|---------------|
| 0s | Input shows blinking cursor + placeholder | Empty canvas |
| 2s | User message types character-by-character | — |
| 5s | AI: "Building your product launch email..." + typing dots | — |
| 6s | — | Eyebrow "New release" slides in |
| 7.5s | — | Heading "Introducing your new workspace" |
| 9s | — | Text paragraph |
| 10.5s | AI: "Added a bold headline and intro copy." | — |
| 12s | — | Gradient image block |
| 14s | — | Two-column features (Automate / Monitor) |
| 15.5s | AI: "Added a product image and feature columns." | — |
| 17s | — | CTA button "Explore the launch" |
| 19s | AI: "Done — your launch email is ready. ✨" | Fully assembled |
| 22–26s | Hold → fade → loop | Hold → fade → loop |

- Uses `setTimeout` chain + Vue refs (no external libs)
- `IntersectionObserver` pauses animation when hero is off-screen
- `prefers-reduced-motion` skips animation, shows final state immediately

### New CSS needed
- `pc-block-enter` — fade-in-up with scale for each email block
- `pc-dot-pulse` — staggered pulsing dots for AI "thinking" indicator
- `pc-cursor` — blinking cursor via `::after` pseudo-element
- `pc-reveal` — initial hidden state for scroll-driven reveals

## 2. Trust Bar
- 5 placeholder company names (Acme, Globex, Initech, Hooli, Pied Piper) as styled text at `opacity-40`
- One testimonial quote below

## 3. Feature Showcase (replaces boring 6-card grid)

3 alternating 2-column sections with real visuals:

**A. "AI builds what you describe"** — mini chat window visual + description
**B. "One email, every client"** — 3 small email preview cards (Gmail/Apple/Outlook) + description  
**C. "Variables and API, built in"** — mini code window with API snippet + description

Plus 3 compact cards below for remaining features (themes, usage metering, pixel-perfect), each with a colored accent bar at top.

Scroll-driven reveal via `IntersectionObserver` adding `pc-rise` class to `.pc-reveal` elements.

## 4. Template Gallery
Horizontal scroll of 6 cards, each with:
- Colored accent strip matching the template's brand color
- Mini email preview (styled divs mimicking the template layout)
- Name + style label

Templates: Product Launch, Editorial Digest, Flash Sale, Warm Welcome, Event Invite, Account Update.

## 5. Final CTA
Same aurora-backed window, stronger copy: "Stop coding email HTML. Start designing."
